# Student Exam Score Predictor

Невеликий навчальний проєкт з машинного навчання та Streamlit.

Мета застосунку — **прогнозувати підсумковий бал студента за іспит/тест** на основі його навчальних звичок:
- годин навчання;
- відвідуваності;
- оцінки ментального стану;
- годин сну;
- факту наявності підробітку.

Проєкт складається з:
- Jupyter-ноутбука для **підготовки даних, навчання моделей та вибору найкращої**;
- простого веб-інтерфейсу на **Streamlit**, який використовує збережену модель (`best_model.pkl`) для прогнозу балу.
